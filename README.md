# EasyRent_Front

### 一个提供在校学生和老师发布租赁商品信息的平台。

[EasyRent_Server](https://github.com/cenganhui/EasyRent_Server)

## 主要功能
- 用户注册登录
- 显示平台中租赁商品信息
- 根据商品标题搜索商品
- 订单模块
- 发布商品和查看已发布商品
- 商品收藏
- 个人信息修改
- 聊天模块
